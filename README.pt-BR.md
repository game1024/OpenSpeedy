# 🚀 Funcionalidades

- Ajuste rápido de velocidade
- Interface moderna
- Suporte para processos x86 e x64
- Sem invasão ao kernel — hooking em Ring-3, não altera o kernel do sistema

# 💾 Instalação

📦 **Método 1: Winget**
```powershell
# Comando de instalação
winget install openspeedy
# Abra um novo terminal e execute
openspeedy
```

📥 **Método 2: Download Manual**
Acesse a [página de Releases](https://github.com/game1024/OpenSpeedy/releases) para baixar a versão mais recente.

# 💻 Requisitos do Sistema

- SO: Windows 10 ou superior
- Plataforma: x86 (32 bits) e x64 (64 bits)

# 📖 Como Usar

1. Inicie o OpenSpeedy
2. Execute o jogo que deseja acelerar
3. Selecione o processo do jogo e ajuste o multiplicador de velocidade na interface do OpenSpeedy
4. O efeito é imediato

# 🔧 Como Funciona

Pré-requisitos:
- [Node.js](https://nodejs.org/) 18+
- [Rust](https://www.rust-lang.org/)
- [CMake](https://cmake.org/)
- [Visual Studio](https://visualstudio.microsoft.com/) (com workload de desenvolvimento desktop em C++)

Comando de build:
```powershell
npm run tauri dev
```

O OpenSpeedy ajusta a velocidade do jogo fazendo hook nas seguintes funções de tempo do sistema Windows:

| Função | Biblioteca | Finalidade |
|--------|-----------|-----------|
| Sleep | user32.dll | Pausa da thread |
| SetTimer | user32.dll | Cria timers baseados em mensagens |
| timeGetTime | winmm.dll | Obtém tempo de atividade do sistema em ms |
| GetTickCount | kernel32.dll | Obtém tempo de atividade do sistema em ms |
| GetTickCount64 | kernel32.dll | Obtém tempo de atividade do sistema em ms (64 bits) |
| QueryPerformanceCounter | kernel32.dll | Contador de performance de alta resolução |
| GetSystemTimeAsFileTime | kernel32.dll | Obtém a hora do sistema |
| GetSystemTimePreciseAsFileTime | kernel32.dll | Obtém a hora do sistema de alta precisão |
| SetWaitableTimer | kernel32.dll | Configura um timer aguardável |
| SetWaitableTimerEx | kernel32.dll | Configura um timer aguardável (estendido) |

# ⚠️ Avisos

- Esta ferramenta é apenas para fins educacionais e de pesquisa
- Alguns jogos online possuem sistemas anti-trapaça — o uso desta ferramenta pode resultar em banimento da conta
- Velocidade excessiva pode causar falhas no motor físico ou travamentos
- Não recomendado para uso em jogos competitivos online
